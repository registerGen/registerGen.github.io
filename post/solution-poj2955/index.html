题解 POJ 2955 【Brackets】

发布于 2020-04-01 | 分类于 区间 DP 、 DP 、 题解 | 4分钟 | 695字数

题目链接：here

在 POJ 闲逛时看到的一道题。

当时正好在学 DP，我也就做了。

题目大意

给你一个由 (，)，[，] 组成的字符串 sss，你要取出这个字符串的一个子序列（即可以不连续），使得它是一个合法括号串（可以是空串）。问最长长度是多少。

多组数据，以 end 结尾。

数据范围，∣s∣≤100|s|\le 100∣s∣≤100，O(∣s∣3)O(|s|^3)O(∣s∣3) 可过。

解法

区间 DP。

和CF607B有点像，我在最后会附上此题的代码。

有点套路（？）

令 fl,rf_{l,r}fl,r​ 为从 lll 到 rrr 的子串中能取出的子序列的长度最大值。

转移方程：

fl,r={max⁡(fl,r,fl+1,r−1+2)if sl=srmax⁡(fl,r,max⁡l≤k<r(fl,k+fk+1,r))f_{l,r}=\begin{cases}\max(f_{l,r},f_{l+1,r-1}+2)&\text{if }s_l=s_r\\\max\left(f_{l,r},\max\limits _{l\le k<r}(f_{l,k}+f_{k+1,r})\right)\end{cases} fl,r​=⎩⎨⎧​max(fl,r​,fl+1,r−1​+2)max(fl,r​,l≤k<rmax​(fl,k​+fk+1,r​))​if sl​=sr​

应该不太难理解。

上面的式子成立是因为 sss 是合法的 ⇒(s)\Rightarrow (s)⇒(s) 是合法的。

下面的式子成立是因为 s,ts,ts,t 是合法的 ⇒st\Rightarrow st⇒st 是合法的。

但这题有个坑点：初始化是全为 000，而不是通常的 fi,i=1f_{i,i}=1fi,i​=1，因为一个字符组成的字符串不可能是合法的！

代码

#include<cstdio>
#include<algorithm>
#include<cstring>

const int N=100;

char s[N+10];
int f[N+10][N+10];

int main()
{
	while(1)
	{
		// 一定记得初始化
		memset(s,0,sizeof(s));
		memset(f,0,sizeof(f));
		scanf("%s",s+1);
		if(s[1]=='e')break;
		int n=strlen(s+1);
		for(int len=1;len<=n;len++) // 枚举长度
			for(int i=1,j=len+1;j<=n;i++,j++) // 枚举区间左右端点
			{
				// 转移
				if((s[i]=='('&&s[j]==')')||(s[i]=='['&&s[j]==']'))
					f[i][j]=std::max(f[i][j],f[i+1][j-1]+2);
				for(int k=i;k<j;k++)
					f[i][j]=std::max(f[i][j],f[i][k]+f[k+1][j]);
			}
		// 答案即为 f[1][n]
		printf("%d\n",f[1][n]);
	}
	return 0;
}

附

CF607B的代码：

#include<cstdio>
#include<algorithm>
#include<cstring>

const int N=500;

int a[N+10],f[N+10][N+10];

int main()
{
	int n;
	scanf("%d",&n);
	for(int i=1;i<=n;i++)
		scanf("%d",a+i);
	memset(f,0x3f,sizeof(f));
	for(int i=1;i<=n;i++)
		f[i][i]=1;
	for(int i=1;i<n;i++)
		f[i][i+1]=1+(a[i]!=a[i+1]);
	for(int len=2;len<=n;len++)
		for(int i=1,j=len+1;j<=n;i++,j++)
		{
			if(a[i]==a[j])f[i][j]=f[i+1][j-1];
			for(int k=i;k<j;k++)
				f[i][j]=std::min(f[i][j],f[i][k]+f[k+1][j]);
		}
	printf("%d\n",f[1][n]);
	return 0;
}

CF383C\color{red}\textsf{CF383C}CF383C

这题的 DFS 序应该这样处理：

void DFS(int u,int fa)
{
	dep[u]=dep[fa]+1;
	st[u]=++cnt;
	for(int i=head[u];i;i=e[i].nxt)
	{
		int v=e[i].to;
		if(v==fa)continue;
		DFS(v,u);
	}
	ed[u]=cnt;
}

ststst 是第一次访问时的编号，ededed 是第二次访问时的编号。

这题是区间修改单点查询，可以用 BIT + 差分。

差分时是修改 stust_ustu​ 和 edu+1ed_u+1edu​+1 的值

而这个点进行修改操作的时候，会与其深度的奇偶性有关。只需判断一下深度，决定修改的值的正负就好。

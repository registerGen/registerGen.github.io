题解 UVA10214 【树林里的树 Trees in a Wood.】

发布于 2020-06-24 | 分类于 莫反 、 数学/数论 、 题解 | 3分钟 | 452字数

莫反模板题。

Code

我们先只考虑第一象限的情况。

简化版题意：给定 n,mn,mn,m，求

∑i=1n∑j=1m[gcd⁡(i,j)=1]\sum_{i=1}^n\sum_{j=1}^m [\gcd(i,j)=1] i=1∑n​j=1∑m​[gcd(i,j)=1]

对上式进行莫反，得

∑i=1n∑j=1m∑d∣gcd⁡(i,j)μ(d)\sum_{i=1}^n\sum_{j=1}^m\sum_{d|\gcd(i,j)}\mu(d) i=1∑n​j=1∑m​d∣gcd(i,j)∑​μ(d)

将第三个 ∑\sum∑ 展开，得

∑i=1n∑j=1m∑d∣i∧d∣jμ(d)\sum_{i=1}^n\sum_{j=1}^m\sum_{d|i\land d|j}\mu(d) i=1∑n​j=1∑m​d∣i∧d∣j∑​μ(d)

改变枚举顺序，得

∑d=1min⁡(n,m)μ(d)⌊nd⌋⌊md⌋\sum_{d=1}^{\min(n,m)}\mu(d)\left\lfloor\frac nd\right\rfloor\left\lfloor\frac md\right\rfloor d=1∑min(n,m)​μ(d)⌊dn​⌋⌊dm​⌋

然后就可以 O(n)O(\sqrt n)O(n​) 数论分块求答案 ansansans 了。

注意：

  1. ansansans 只是一个象限的答案，实际的答案（分子）为 4ans+44ans+44ans+4（444 个象限 + 坐标轴）。
  2. 要去掉原点，所以分母为 (2n+1)(2m+1)−1(2n+1)(2m+1)-1(2n+1)(2m+1)−1
  3. 要开 long long。

Code

#include<cstdio>
#include<algorithm>

typedef long long ll;

const int N=2000;

int n,m,prime[N+10],mu[N+10],notPrime[N+10],cnt;
ll ans,tot;

inline void init()
{
	mu[1]=1;
	for(int i=2;i<=N;i++)
	{
		if(!notPrime[i])prime[++cnt]=i,mu[i]=-1;
		for(int j=1;j<=cnt&&i*prime[j]<=N;j++)
		{
			notPrime[i*prime[j]]=1;
			if(i%prime[j]==0)break;
			mu[i*prime[j]]=-mu[i];
		}
		mu[i]+=mu[i-1];
	}
}

int main()
{
	init();
	while(scanf("%d%d",&n,&m)!=EOF)
	{
		if(!n)return 0;
		if(n>m)std::swap(n,m);
		ans=0;
		for(int l=1,r;l<=n;l=r+1)
		{
			r=std::min(n/(n/l),m/(m/l));
			ans+=1LL*(mu[r]-mu[l-1])*(n/l)*(m/l);
		}
		ans=ans*4LL+4LL;
		tot=1LL*(n*2+1)*(m*2+1)-1;
		printf("%.7lf\n",1.0*ans/tot);
	}
	return 0;
}

Solution

出题人 tql！我来说一个 O(Tlog⁡n)O(T\log n)O(Tlogn) 的解法。

首先特判一下：

  • 若 m≤1m\le 1m≤1 或 m>nm>nm>n，则答案为 000。
  • 若 m=n−1m=n-1m=n−1 或 m=nm=nm=n，则答案为 111。

突破点：

当 i≤ji\le ji≤j 时，⌊ij⌋+⌈ji⌉=⌈ji⌉\left\lfloor\dfrac{i}{j}\right\rfloor+\left\lceil\dfrac{j}{i}\right\rceil=\left\lceil\dfrac{j}{i}\right\rceil⌊ji​⌋+⌈ij​⌉=⌈ij​⌉；

当 i≥ji\ge ji≥j 时，⌊ij⌋+⌈ji⌉=⌊ij⌋+1\left\lfloor\dfrac{i}{j}\right\rfloor+\left\lceil\dfrac{j}{i}\right\rceil=\left\lfloor\dfrac{i}{j}\right\rfloor+1⌊ji​⌋+⌈ij​⌉=⌊ji​⌋+1。

所以我们就把本题转换成了这题：

给定 n,mn,mn,m，求有多少组 (i,j)(i,j)(i,j) 满足

{i+j=n⌈ji⌉=m 或 ⌊ij⌋=m−1\begin{cases}i+j=n\\\left\lceil\dfrac{j}{i}\right\rceil=m\text{ 或 }\left\lfloor\dfrac{i}{j}\right\rfloor=m-1\end{cases} ⎩⎨⎧​i+j=n⌈ij​⌉=m 或 ⌊ji​⌋=m−1​

很明显 ⌈ji⌉\left\lceil\dfrac{j}{i}\right\rceil⌈ij​⌉ 和 ⌊ij⌋\left\lfloor\dfrac{i}{j}\right\rfloor⌊ji​⌋ 有单调性，于是二分一（四）下就好了。

Code

#include<cstdio>
#include<algorithm>
#include<cmath>

int T,n,m;

int bin1()
{
	int l=1,r=n;
	while(l<=r)
	{
		int mid=(l+r)>>1;
		if(int(ceil(1.0*mid/(n-mid)))>m)r=mid-1;
		else l=mid+1;
	}
	return r;
}

int bin2()
{
	int l=1,r=n;
	while(l<=r)
	{
		int mid=(l+r)>>1;
		if(int(ceil(1.0*mid/(n-mid)))>=m)r=mid-1;
		else l=mid+1;
	}
	return r;
}

int bin3()
{
	int l=1,r=n;
	while(l<=r)
	{
		int mid=(l+r)>>1;
		if(int(floor(1.0*mid/(n-mid)))>=m)r=mid-1;
		else l=mid+1;
	}
	return r;
}

int bin4()
{
	int l=1,r=n;
	while(l<=r)
	{
		int mid=(l+r)>>1;
		if(int(floor(1.0*mid/(n-mid)))>=m-1)r=mid-1;
		else l=mid+1;
	}
	return r;
}

int main()
{
	scanf("%d",&T);
	while(T--)
	{
		scanf("%d%d",&n,&m);
		if(m<=1||m>n){puts("0");continue;}
		if(m==n-1||m==n){puts("1");continue;}
		int ans=0;
		ans+=bin1()-bin2();
		ans+=bin3()-bin4();
		printf("%d\n",ans);
	}
	return 0;
}

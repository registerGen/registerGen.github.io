Miller-Rabin 素数测试学习笔记

2020-04-01 题解 学习笔记

众所周知，判断一个数是否是素数有一个 O(n)O(\sqrt{n})O(n​) 的算法：

inline bool isPrime(ll n)
{
	if(n<=1)return false;
	for(int i=2;i<=sqrt(n);i++)
		if(n%i==0)return false;
	return true;
}

那么，有没有什么更快的算法呢？

那就是 Miller-Rabin 素数测试！
